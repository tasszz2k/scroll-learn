Scroll · Learn

Install the quiet feed.

ScrollLearn injects spaced-repetition flashcards into Facebook, YouTube, and Instagram feeds. macOS + Chrome.

↓ Download installer
curl -fsSL https://raw.githubusercontent.com/tasszz2k/scroll-learn/main/install.sh | bash copied
After clicking download
  1. Open Downloads in Finder, right-click install.command → Open (the first time only; macOS warns about unsigned scripts).
  2. A Terminal window opens and runs the installer. It downloads the latest release into ~/.scroll-learn/.
  3. The installer will open Chrome at chrome://extensions. Toggle Developer mode, click Load unpacked, pick ~/.scroll-learn/extension.
  4. Copy the extension ID from the ScrollLearn card and paste it back into Terminal.
  5. Reload the extension once. Done. Future updates appear as a one-click banner.

Updates from now on

The dashboard shows a banner when a new release is out. Click Update now, the new version installs in a few seconds, and the extension reloads. No terminal, no downloads.
